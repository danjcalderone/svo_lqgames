# svo_lqgames
Code for paper "Analysis of Nash Equilibria in Open-Loop Linear Quadratic Games for Trajectory Planning with Social Value Orientation"

## Requirements

- Python 3.x
- Required packages (if any)

Install dependencies with:

```bash
pip install -r requirements.txt
```

## Repository Structure

```
.
├── SVO_MAIN.ipynb              # Main jupyter notebook to run
├── svo_classes.py    # Supporting functions and utilities
├── svo_examples.py    # Sets up and runs simulations
├── svo_frames.py    # Plotting functions
├── svo_styles.py    # Generates plotting style information
├── requirements.txt     # Python dependencies
├── README.md            # Project documentation
└── .gitignore           # Git ignore rules
```

## Running the Code

Run the main program from jupyter notebook SVO_MAIN.ipynb


## Source Files


- **SVO_MAIN.ipynb** – Main jupyter notebook to run
- **svo_classes.py** – Supporting functions and utilities
- **svo_examples.py** – Sets up and runs simulations
- **svo_frames.py** – Plotting functions
- **svo_styles.py** – Generates plotting style information

## Output

The output is shown as interactive plots using ipywidgets in SVO_MAIN.ipynb
